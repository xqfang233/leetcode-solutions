### Question
Given an integer array nums, return true if any value appears at least twice in the array, and return false if every element is distinct.


### Examples
Example 1:

Input: nums = [1,2,3,1]
Output: true
Example 2:

Input: nums = [1,2,3,4]
Output: false
Example 3:

Input: nums = [1,1,1,3,3,4,3,2,4,2]
Output: true


### Constraints
1 <= nums.length <= 105
-109 <= nums[i] <= 109


### Solutions

#### Solution 1: set
```
class Solution:
    def containsDuplicate(self, nums: List[int]) -> bool:
        unique_set = set()
        for num in nums:
            if num in unique_set:
                return True
            else:
                unique_set.add(num)
        return False
```
time complexity: O(n)
space complexity: O(n)

#### Solution 2: compare the length of unique set and the list
```
class Solution:
    def containsDuplicate(self, nums: List[int]) -> bool:
        # compare the length of unique set and the list
        return len(set(nums))!=len(nums)
```
time complexity: O(n)
space complexity: O(n)


#### bit manipulation
```
class Solution:
    def containsDuplicate(self, nums: List[int]) -> bool:
        # use left shift to store the number traversed
        # use OR to set the corresponding bit to 1
        # use AND to check if the new comer already existed
        # need to consider positive and negative integers separately
        n_checker = 0 # for negative integers
        p_checker = 0 # for positive integers
        
        for num in nums:
            if num >= 0:
                if (p_checker & 1 << num) >0:
                    return True
                p_checker |= (1 << num)
            
            if num <0:
                num = abs(num)
                if (n_checker & 1 << num) > 0:
                    return True
                n_checker |= (1 << num)
        return False
```
time complexity: O(n)
space complexity: O(1)
